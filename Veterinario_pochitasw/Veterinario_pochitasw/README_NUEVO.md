# Sistema Veterinario "Pochita S.A."

## Estructura de Carpetas

*   **backend/**: Contiene el código fuente de Django (`manage.py`, `pochita_project`, `clinic`, etc).
*   **frontend/**: Contiene los templates HTML y archivos estáticos (CSS/JS).
*   **base_de_datos/**: Contiene la base de datos SQLite (`db.sqlite3`).

## Cómo Ejecutar

1.  Abrir terminal.
2.  Entrar a backend:
    `cd backend`
3.  Ejecutar:
    `python manage.py runserver`

El sistema detectará automáticamente los templates y base de datos en las carpetas hermanas.
